# Da Krunch

---

# Details

**Witchfire:** _Da Krunch_ has a warp charge value of 6. If manifested, select one enemy unit that is within 18" of this PSYKER. Roll one D6 for each model in that unit; for each roll of 6, that enemy unit suffers 1 mortal wound. If the result of the Psychic test was 9 or more, add 1 to rolls when resolving this psychic power.
